## 📞 遇到问题？

### 自助排查

1. **查文档** - 看《AI协作编程指南》的"常见问题解答"
2. **看控制台** - 按F12，截图红色错误
3. **恢复备份** - 用备份文件恢复
4. **重新提问** - 用模板重新描述需求

### 何时需要重新描述需求

- ❌ AI改了好几次还是不对
- ❌ 功能完全不能用
- ❌ 页面布局乱了
- ❌ 影响了其他功能

**这时：**
1. 恢复备份
2. 用模板重新描述需求
3. 强调"不能影响XX功能"
4. 要求AI先提供方案

## 🎯 核心要点

### 记住这个流程

```
复制模板 → 填入需求 → 发给AI → 等待修改 → 测试验收 → 通过/重做
```

### 记住这5句话

1. **说效果，不说代码**
2. **说清楚不能影响什么**
3. **要求白话解释**
4. **要求测试步骤**
5. **要求撤销方法**

### 记住这个心态

```
你是产品经理，AI是程序员。
你描述需求，AI负责实现。
你验收质量，AI负责修复。
```
