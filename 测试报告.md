## 📝 使用说明

### 如何更新国家数据
1. 打开 `index.html` 文件
2. 搜索 `📊 数据配置区`
3. 找到对应的数据数组（如 `visaFreeCountries`）
4. 按照现有格式添加/修改国家
5. 保存文件，刷新浏览器即可

### 如何修改政策文本
1. 打开 `index.html` 文件
2. 搜索 `🌍 国际化数据区`
3. 找到 `translations` 对象
4. 修改对应的中文（`zh`）或英文（`en`）文本
5. 保存文件，刷新浏览器即可

### 如何添加新功能模块
1. 在HTML部分找到对应的大区（如核心工具区）
2. 添加新的HTML结构
3. 在JavaScript部分找到对应的函数区
4. 添加新的函数逻辑
5. 在初始化区添加调用代码
6. 保存文件，测试功能
