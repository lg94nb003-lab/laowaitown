## 核心工具区

### 1. 签证计算器

**功能描述：** 计算签证最晚离境日期

#### 主要特性

- 📅 **日期输入** - 支持年月日输入，可用鼠标滚轮调整
- ⚡ **快捷按钮** - 提供"今天"、"昨天"、"明天"等快捷选项
- 🔢 **天数快捷** - "30天"、"60天"、"180天"快捷按钮
- 📤 **结果复制** - 计算结果可一键复制

#### 使用场景

```
场景：你持旅游签证入境中国，想知道最晚什么时候必须离境

操作步骤：
1. 输入入境日期（如：2024年1月1日）
2. 输入可停留天数（如：30天）
3. 点击"立即计算"
4. 显示最晚离境日期（如：2024年1月30日）
5. 点击复制按钮保存结果
```

#### 界面位置

页面顶部 - 核心工具区左侧

---

### 2. 时区转换器

**功能描述：** 在不同时区之间转换时间

#### 主要特性

- 🌍 **多时区支持** - 支持全球主要城市和时区
- 🏙️ **快捷城市** - "北京"、"伦敦"、"纽约"等常用城市
- ⏰ **时间偏移** - "-24h"、"现在"、"+24h"快捷按钮
- 📤 **结果复制** - 转换结果可一键复制

#### 使用场景

```
场景：你在纽约，需要知道北京时间下午3点对应纽约什么时间

操作步骤：
1. 输入源时间（如：2024年1月1日 15:00）
2. 选择源时区（北京）
3. 选择目标时区（纽约）
4. 点击"转换时区"
5. 显示转换后的时间（如：2024年1月1日 02:00）
```

#### 界面位置

页面顶部 - 核心工具区中间

---

### 3. 汇率换算器

**功能描述：** 实时汇率换算

#### 主要特性

- 💱 **多货币支持** - 支持CNY、USD、EUR、JPY等主要货币
- 📊 **实时汇率** - 调用外部API获取最新汇率
- ⚡ **快捷金额** - "500"、"2000"、"5000"、"10000"快捷按钮
- 🔄 **货币交换** - 点击双向箭头快速交换货币
- 📤 **结果复制** - 换算结果可一键复制

#### 使用场景

```
场景：你有1000美元，想知道能兑换多少人民币

操作步骤：
1. 输入金额（如：1000）
2. 选择"从"货币（USD）
3. 选择"到"货币（CNY）
4. 点击"获取汇率"
5. 显示兑换结果（如：7200 CNY）
```

#### 界面位置

页面顶部 - 核心工具区右侧

## 免签政策区

### 4. 免签入境

**功能描述：** 查询50个国家的免签政策

#### 主要特性

- 🌍 **50个国家** - 涵盖欧洲35个、亚洲7个、美洲6个、大洋洲2个
- 🗺️ **大洲筛选** - 可按"全部"、"欧洲"、"亚洲"、"美洲"、"大洋洲"筛选
- 🔍 **智能搜索** - 支持国家名称（如"德国"）和国家代码（如"DE"）搜索
- 🎨 **颜色标识** 
  - 绿色政策框：50个指定免签国家
  - 红色提示框：其他国家
- 📱 **响应式布局** - 自适应不同屏幕尺寸

#### 使用场景

```
场景：你是法国公民，想知道是否需要签证来中国

操作步骤：
1. 点击"免签入境"标签
2. 点击"欧洲"筛选（或直接搜索"法国"）
3. 点击"法国"方块
4. 显示绿色政策框，说明免签政策
5. 查看详细停留时间和允许活动
```

#### 数据配置

- 数据位置：`📊 数据配置区` → `visaFreeCountries` 数组
- 包含字段：国家代码(code)、中文名(zh)、英文名(en)、所属大洲(continent)

#### 界面位置

页面中部 - 免签政策区第一个标签页

### 5. 240h过境免签

**功能描述：** 查询54个国家的240小时（10天）过境免签政策

#### 主要特性

- 🌍 **54个国家** - 覆盖主要发达国家
- 🛃 **30个口岸** - 支持多个入境口岸选择
- 📋 **详细政策** - 显示免签停留时间和允许活动范围
- ⚠️ **条件提醒** - 提醒必备条件（如持第三国机票）

#### 使用场景

```
场景：你是美国公民，从中国转机去日本，想利用过境免签

操作步骤：
1. 点击"240h 过境免签"标签
2. 选择国籍（美国）
3. 选择入境口岸（如：北京首都机场）
4. 显示免签政策详情
5. 查看允许停留时间和活动范围
```

#### 数据配置

- 国家数据：`📊 数据配置区` → `transit240Countries` 数组
- 省份/口岸数据：`📊 数据配置区` → `transit240Provinces` 数组

#### 界面位置

页面中部 - 免签政策区第二个标签页

## 国际化支持

### 中英文切换

**功能描述：** 完整的中英文双语支持

#### 主要特性

- 🌐 **一键切换** - 点击顶部"EN"或"中文"按钮
- 📝 **完整翻译** - 所有界面元素、政策文本、提示信息
- 🔄 **无缝切换** - 切换后所有模块同步更新
- 💾 **状态保持** - 切换语言后保持当前操作状态

#### 使用场景

```
场景：你的中文不好，想切换到英文界面

操作步骤：
1. 点击页面顶部的"EN"按钮
2. 所有文字切换为英文
3. 正常使用所有功能
4. 需要时点击"中文"切换回来
```

#### 技术实现

- 数据位置：`🌍 国际化数据区` → `translations` 对象
- 包含：zh（中文）和 en（英文）两个语言包
- 切换机制：通过 JavaScript 动态替换文本内容
