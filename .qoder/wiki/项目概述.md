## 项目背景

### 为什么做这个项目？

随着中国对外开放程度的不断提高，越来越多的外国人选择来中国工作、学习、旅游或生活。然而，由于语言障碍和文化差异，外国人在中国生活中常常遇到以下困难：

1. **签证政策复杂** - 免签政策、过境免签、签证计算器等难以理解
2. **时区转换困难** - 与中国存在时差，需要频繁转换时间
3. **汇率换算不便** - 货币兑换计算复杂
4. **生活服务不熟悉** - 支付方式、外卖平台、出行工具等不了解
5. **法律合规不清楚** - 住宿登记、工作许可等规定不熟悉

### 项目目标

**waiguoren1** 项目旨在为外国人提供一个**一站式**的中国生活服务查询平台，帮助他们：

- ✅ 快速了解中国签证政策
- ✅ 方便地进行时区和汇率计算
- ✅ 熟悉中国的生活服务
- ✅ 遵守中国的法律法规
- ✅ 顺利适应中国的生活

## 技术栈

### 核心技术

这是一个**纯前端单页应用**，不依赖任何后端服务：

| 技术 | 用途 | 说明 |
|------|------|------|
| **HTML5** | 页面结构 | 语义化标签，清晰的分区结构 |
| **CSS3** | 页面样式 | 响应式设计，现代化UI |
| **JavaScript** | 交互逻辑 | 原生JS，无框架依赖 |

### 技术特点

#### 优点

- ✅ **零依赖** - 不需要安装任何依赖包
- ✅ **轻量级** - 单个 HTML 文件，约 277KB
- ✅ **易部署** - 可以直接在浏览器中打开使用
- ✅ **易维护** - 代码分区清晰，注释完善
- ✅ **高性能** - 无框架开销，加载快速

#### 限制

- ⚠️ **数据静态** - 汇率等数据需要手动更新或调用外部API
- ⚠️ **无数据库** - 所有数据存储在代码中
- ⚠️ **单文件** - 所有代码在一个 HTML 文件中

## 项目特点

### 1. 代码结构优化

项目经过系统性优化，采用**清晰的分区架构**：

```
代码结构 = 一套房子
├── 🎨 样式区 = 装修材料间
├── 🌐 页面主体区 = 客厅
│   ├── 🛠️ 核心工具区
│   ├── 🛂 免签政策区
│   └── 🏠 生活在中国区
└── 💻 JavaScript代码区 = 大脑中枢
    ├── 📊 数据配置区（改数据看这里）
    └── 各种函数区（不要动）
```

**优势：**
- 找东西快：想找"国家列表"？直接去"数据配置区"
- 修改安全：改"数据配置区"不会影响"工作流程区"
- 新人友好：看门牌就知道每个区域是干嘛的

### 2. 数据与逻辑分离

- **数据配置区** - 集中管理所有数据（国家列表、政策文本等）
- **代码逻辑区** - 处理业务逻辑和交互

**好处：**
- 更新数据：只改"数据配置区"，1分钟搞定
- 不会出错：不动"代码逻辑区"，功能不受影响
- 职责清晰：数据归数据，逻辑归逻辑

### 3. 全中文注释

所有代码注释都使用**中文**，方便非英语母语的开发者理解：

```javascript
// 之前（英文注释）
// Select country
function selectVfCountry(code) { ... }

// 现在（中文注释）
// 选中免签国家并填充详情
function selectVfCountry(code) { ... }
```

### 4. AI协作友好

项目专门为**非技术人员与AI协作**而优化：

- 清晰的分区标记，方便定位代码
- 数据配置区集中，修改简单
- 完善的文档体系，降低学习成本
- 标准化的提问模板，提高协作效率
