## 维护操作指南

### 如何更新国家数据

#### 添加免签国家

**步骤：**

```
1. 打开 index.html 文件
   （使用代码编辑器，如VS Code）

2. 搜索"📊 数据配置区"
   （Ctrl + F 搜索）

3. 找到 visaFreeCountries 数组

4. 在对应大洲区域添加新国家：
   
   // 亚洲国家示例
   { code: 'VN', zh: '越南', en: 'Vietnam', continent: 'asia' },

5. 保存文件（Ctrl + S）

6. 刷新浏览器（Ctrl + F5）

7. 测试：
   - 点击"免签入境"
   - 点击对应大洲标签
   - 查看新国家是否显示
   - 点击新国家查看政策
```

**注意事项：**
- 按照现有格式添加
- 国家代码必须是2个大写字母
- continent 必须是：europe/asia/americas/oceania 之一
- 不要忘记末尾的逗号

---

#### 添加过境口岸

**步骤：**

```
1. 打开 index.html 文件

2. 搜索"📊 数据配置区"

3. 找到 `transit240Provinces` 数组（如果是添加适用国家则是 `transit240Countries`）

4. 添加新口岸：
   
   { 
       code: 'XXX', 
       zh: '口岸中文名', 
       en: 'Port English Name',
       city: '所在城市'
   },

5. 保存文件，刷新浏览器

6. 测试：
   - 点击"240h 过境免签"
   - 查看口岸下拉框
   - 选择新口岸测试
```

---

### 如何修改政策文本

#### 修改免签政策文本

**步骤：**

```
1. 打开 index.html 文件

2. 搜索"🌍 国际化数据区"

3. 找到 translations 对象

4. 修改中文文本：
   
   zh: {
       vfEntryPolicyText: '新的中文政策文本...',
   }

5. 修改英文文本：
   
   en: {
       vfEntryPolicyText: 'New English policy text...',
   }

6. 保存文件，刷新浏览器

7. 测试：
   - 选择免签国家（如德国）
   - 查看政策文本是否更新
   - 切换到英文查看英文版本
   - 切换回中文查看中文版本
```

**注意事项：**
- 保持引号匹配
- 注意转义字符（如需要）
- 中英文都要修改
- 测试两种语言

---

### 如何添加新功能模块

#### 添加新模块的完整流程

**步骤：**

```
第1步：规划
- 确定模块功能
- 确定放置位置
- 设计样式风格

第2步：HTML结构
- 在页面主体区找到对应大区
- 添加新的HTML结构
- 使用清晰的分区注释

第3步：CSS样式
- 在组件样式区添加新样式
- 保持与现有风格一致
- 添加响应式支持

第4步：JavaScript逻辑
- 在功能函数区添加新函数
- 在数据配置区添加数据（如需要）
- 在初始化区添加调用代码

第5步：国际化
- 在 translations 中添加中英文翻译
- 测试语言切换

第6步：测试
- 测试新功能
- 测试所有现有功能
- 检查控制台无错误
```

**示例：添加"紧急联系方式"模块**

```html
<!-- 在🏠 生活在中国区添加 -->
<div class="emergency-contacts">
    <h3>紧急联系方式</h3>
    <div class="contact-item">
        <span class="label">报警电话：</span>
        <span class="number" onclick="copyText('110')">110</span>
    </div>
    <!-- 更多联系方式 -->
</div>
```

```javascript
// 在功能函数区添加
function copyText(text) {
    navigator.clipboard.writeText(text);
    alert('已复制：' + text);
}
```
