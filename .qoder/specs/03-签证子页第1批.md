## 用户故事

外国人通过 Google 搜索"240-hour transit visa china" / "hainan visa-free" 等长尾词时，能直接落地到一个**专题深度页**（不是首页），页面要：
1. 第一屏给出 Quick Answer，无需滚动即知是否符合条件；
2. 显示**政策状态徽章**（绿色 Active / 蓝色 Recently updated / 红色 Outdated），让访客一眼判断信息时效；
3. 提供完整的国家清单、口岸清单、申请步骤、常见误区、FAQ；
4. 给一个跳回首页对应交互工具的 CTA（"Check eligibility →"）；
5. 信源可见但不挂超链接（防死链 + 防跨境访问问题）；
6. 移动端 60% 流量必须可用（响应式）。

## 数据契约（写新页时按这个填）

### `LWPolicyStatus` 条目模板

```js
'<pageId>': {
    /* 可选：手动覆盖状态。不写则按 effectiveSince 自动判定 */
    // status: 'outdated',

    /* 必填：政策生效日期。决定徽章颜色（≤365 天蓝、>365 天绿） */
    effectiveSince: 'YYYY-MM-DD',

    /* 可选：变更日志。空数组则隐藏 What's new 区块 */
    changelog: [
        { date: 'YYYY-MM-DD', text: 'Short change description', isNew: true },
        { date: 'YYYY-MM-DD', text: 'Older change' }
    ],

    /* 必填：信源。纯文字数组，禁止挂超链接 */
    sources: [
        'National Immigration Administration (NIA)',
        'China Consular Service'
    ]
}
```

## 每页 HTML 骨架模板（直接照搬 240h 改）

参考样板：[visa/240-hour-transit-visa.html](file:///d:/AIdocument/lg1/waiguoren1/visa/240-hour-transit-visa.html)

每页必须包含的结构（按顺序）：

1. `<head>` — GA4 / SEO meta（title / description / keywords / canonical / og / hreflang）/ JSON-LD Article / 引用 `../style.css` `../shared/page.css` `Inter` 字体
2. `<nav class="lw-nav">` — 顶部固定导航，含 logo / `<ul id="lw-nav-links">`（JS 渲染）/ 主题切换按钮
3. `<div class="lw-layout">` 三栏：
   - `<aside id="lw-side-cats">` 左侧同分类清单（JS 渲染）
   - `<article class="lw-article">` 中间正文
   - `<aside class="lw-toc">` 右侧 On this page TOC（手写 anchor link）
4. 正文内部按顺序：
   - `<nav class="lw-breadcrumb">` 面包屑
   - `<h1 id="top">标题<span id="lw-status-pill" class="lw-status-pill" style="display:none"></span></h1>`
   - `<div class="lw-meta">` 元信息（read time / author）
   - `<div id="lw-changelog" class="lw-changelog" style="display:none"></div>` What's new 占位
   - `<aside class="lw-quick">` Quick Answer
   - 多个 `<h2>` 章节（**每个 h2 必须有 id，方便右侧 TOC 跳转**）
   - 至少 1 个 `<div class="lw-table-wrap"><table>` 数据表
   - 1 个 `<div class="lw-phrase">` 中文实用句框
   - 至少 1 个 `<div class="lw-cta">` 跳回首页工具的 CTA
   - `<div class="lw-faq">` FAQ（5-6 个 `<details>`）
   - `<div class="lw-feedback">` 反馈按钮
   - `<section class="lw-related">` 相关阅读（4 个其它签证页）
5. `<div id="lw-sources" class="lw-sources" style="display:none"></div>` 信源占位
6. `<footer class="lw-footer">` 页脚
7. `<script src="../shared/nav-data.js">` + `DOMContentLoaded` 内调用 5 个 init 函数：
   ```js
   window.LWRenderNav('visa');
   window.LWRenderSideCats('visa', '/visa/<this-file>.html');
   window.LWRenderStatus('<pageId>');
   window.LWInitTheme();
   window.LWInitTocSpy();
   window.LWInitFeedback('<pageId>');
   ```

## 日常开发须知

- 每写完一页 HTML，**双击在浏览器打开自查一遍**（无需服务器）
- 写正文时，**所有数字（国家数 / 口岸数 / 天数）以官方最新公告为准**，错了用户会投诉
- Chinese phrase 框的拼音必须正确（拿不准就在 nav-data.js 里加注释或问用户）
- 状态徽章颜色由 `effectiveSince` 自动判定，**不要手写 status 字段**，除非政策真的废止
- commit 之前先 `git status` 看一下变更范围，避免误提交无关文件
- S7 重新编译命令：
  ```powershell
  Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass ;
  npx tailwindcss -i ./src/input.css -o ./style.css --minify
  ```
